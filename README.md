# CHF ckEditor
Bindet den ckEditor (derzeit in Version 4.2) in den Oxid E-Shop ein.
Für OXID CE 4.7.x
## Installation
1. Kopiere chf_ckeditor in dein modules Verzeichnis
2. Aktiviere das Modul im Backend unter Erweiterungen - Module - CHF ckEditor
3. Fertig
